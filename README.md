## Project Experience 
[**Yanxi Trail - Visual Novel Survival Game**](https://zcxu-4444.github.io/yanxi-trail)

A browser-based survival novel strategy game inspired by a Chinese historical drama, *Story of Yanxi Palace*, and a retro educational video game *The Oregon Trail*. Navigate through palace schemes, avenge your sister and allies, manage stats, and survive 32 years in the Qing Dynasty court. Built with p5.js.

[**Amhrest - Off-Campus Housing Platform**](https://github.com/akoduru727/CS320-Spring2026-S3-Team1)

A full-stack off-campus housing platform for students. Built with Svelte, Tailwind, Node.js, and Supabase, featuring a rental listings, real-time chatroom with roommates, a roommate connect page, and responsive UI.

[**Alpaca Engine — FP Game Engine**](https://github.com/mart-codes-good/3D-Simulation-Engine-Alpaca-Engine-1)

A first-person engine built from scratch in C, featuring real-time 3D rendering, AABB collision detection, hitscan logic, and spatial audio. Powered by OpenGL, GLFW, and OpenAL.
